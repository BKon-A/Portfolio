## Technologies Used

- **Language:** C#
- **Framework:** WPF
- **Data Access:** ADO.NET
- **Reporting:** FastReports
- **Database:** SQL (PostgreSQL)

## Features

- Track pollution data from various sources.
- Generate detailed analytical reports.
- Interactive data visualization.
